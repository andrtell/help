# Linux

## Info

### List OS info
```
$ uname -a
```

### List loaded kernel modules
```
$ lsmod
```

### List CPU info
```
$ lscpu
```

### List hardware info
```
$ sudo lshw -short
```

### List memory info
```
$ lsmem
```

### List block devices
```
$ lsblk
```

### List network devices
```
$ ip link
```

### Show disk usage
```
$ df -h
```

### Show memory usage

```
$ free -h
```
